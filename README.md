# Vikram Enterprise - Medical Management System

A comprehensive medical dashboard and administrative panel built with React, TypeScript, and Vite.

## Features

- **Dynamic Dashboard**: Real-time visualization of medical statistics.
- **Admin Panel**: Manage feedbacks, tickets, and day-wise analysis.
- **Interactive UI**: Rich animations and responsive design.
- **Data Analysis**: Monthly and yearly analysis with charts.

## Tech Stack

- **Frontend**: React, TypeScript, Vite, Tailwind CSS
- **Charts**: Recharts
- **Icons**: Lucide React
- **Backend (API)**: Node.js / Express (in `backend/` directory)

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server:
```bash
npm run dev
```

### Build

Build the production bundle:
```bash
npm run build
```

## Project Structure

- `src/components`: UI components including Dashboard and Admin Panel.
- `backend/`: Backend logic and API endpoints.
- `public/`: Static assets.
